# Biblioteca REST API

Este proyecto es una API REST para gestionar una biblioteca. Permite a los usuarios gestionar libros, realizar préstamos y devolverlos.

## Tecnologías

- Spring Boot
- Spring Data JPA
- Lombok
- MapStruct
- Swagger

  
## Funcionalidades

- CRUD de Libros y Usuarios.
- Gestión de Préstamos.
- Filtros y Paginación de libros.
- Gestión de errores y excepciones personalizadas.

## Instalación y Ejecución

1. Clonar el repositorio.
2. Configurar la base de datos en el archivo `application.properties`.
3. Ejecutar con Maven o tu IDE favorito.
